## 模块功能

```
1) 内置组件-无线滚动列表
2) 内置组件-小程序内嵌H5页面
3) 内置组件-手机号登录
4) 内置组件-picker
5) uni ui — 表单
6) uni ui — 级联选择器
7) uni ui — 上传图片
8) Api-tabBar 设置
9) Api-上传图片（推荐）
10) Api-worker多线程
11) Api-获取地理位置（推荐）
12) uview-plus up-picker（级联）
13) uview-plus up-form（表单）
14) 分包
15) 接口调用
16) pinia（持久化）
17) 业务-按钮

```
